# 성과기여도 평가 학습 시스템

한국보건복지인재원(KOHI)의 성과기여도 평가 학습 시스템입니다. 이 시스템은 교육과정의 성과를 체계적으로 평가하고 관리할 수 있는 웹 기반 플랫폼입니다.

## 🚀 주요 기능

### 📚 이론 학습
- **커크패트릭 4단계 모델** 학습
- **SMART 원칙** 기반 KPI 설정
- **ROI 계산기**를 통한 투자 대비 수익률 분석
- **8개 교육과정**별 상세 예시 및 시나리오

### 🎯 실습 모듈
- **과정별 평가 실습**
- **단계별 성과 측정**
- **실제 시나리오 기반 학습**

### 🛠️ 성과 관리 도구
- **KPI 설정 도구**
- **ROI 계산기**
- **보고서 생성기**
- **실습 시나리오**

## 📱 모바일 최적화

이 웹 페이지는 모바일 환경에서 최적화되어 있습니다:

### ✅ 모바일 최적화 기능

#### 📱 반응형 디자인
- **480px 이하**: 모바일 전용 레이아웃
- **481px-768px**: 태블릿 최적화 레이아웃
- **768px 이상**: 데스크톱 레이아웃

#### 🎯 터치 인터페이스 최적화
- **44px 최소 터치 영역** 보장
- **터치 피드백** 제공 (버튼 클릭 시 시각적 반응)
- **스와이프 제스처** 지원 (좌우 스와이프로 탭 전환)
- **더블 탭 방지** 기능

#### 📝 폼 요소 최적화
- **iOS 줌 방지** (16px 폰트 크기)
- **자동 스크롤** (입력 필드 포커스 시)
- **터치 친화적** 체크박스/라디오 버튼

#### 📊 차트 및 테이블 최적화
- **반응형 차트** (Chart.js 최적화)
- **가로 스크롤** 테이블
- **모바일 친화적** 데이터 표시

#### 🔄 화면 방향 대응
- **가로 모드** 최적화
- **세로 모드** 최적화
- **자동 레이아웃 조정**

### 🎨 접근성 개선
- **고대비 모드** 지원
- **다크 모드** 지원
- **애니메이션 감소** 설정 대응
- **스크린 리더** 지원

### ⚡ 성능 최적화
- **부드러운 스크롤** (`-webkit-overflow-scrolling: touch`)
- **터치 액션 최적화** (`touch-action: manipulation`)
- **이미지 최적화** (반응형 이미지)
- **지연 로딩** 지원

## 🛠️ 기술 스택

### Frontend
- **HTML5** - 시맨틱 마크업
- **CSS3** - 반응형 디자인 및 모바일 최적화
- **JavaScript (ES6+)** - 인터랙티브 기능
- **Chart.js** - 데이터 시각화

### 모바일 최적화
- **CSS Media Queries** - 반응형 레이아웃
- **Touch Events** - 터치 인터페이스
- **Viewport Meta Tags** - 모바일 뷰포트 설정
- **Progressive Web App** 준비

```
└── kpi-tool.js               # KPI 도구 JavaScript
```

## 🚀 사용 방법

### 1. 웹 브라우저에서 실행
```bash
# 파일을 웹 서버에 업로드하거나 로컬 서버에서 실행
# 예: Python 간단 서버
python -m http.server 8000

# 또는 Node.js http-server
npx http-server
```

### 2. 모바일에서 접속
- 모바일 브라우저에서 URL 접속
- 자동으로 모바일 최적화 레이아웃 적용
- 터치 제스처 및 인터페이스 활성화

### 3. 주요 기능 사용
- **이론 탭**: 커크패트릭 모델 학습
- **실습 탭**: 과정별 평가 실습
- **성과 관리 도구 탭**: KPI 설정 및 ROI 계산

## 🔧 커스터마이징

### 모바일 스타일 수정
`mobile-optimization.css` 파일에서 모바일 전용 스타일을 수정할 수 있습니다:

```css
/* 모바일에서 버튼 크기 조정 */
@media (max-width: 480px) {
    .btn {
        min-height: 48px !important;
        font-size: 1rem !important;
    }
}
```

### 모바일 JavaScript 기능 추가
